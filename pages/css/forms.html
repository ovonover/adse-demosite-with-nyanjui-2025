HTML Forms

An HTML Form is an interactive tool used in your webpage that is made up of various controls used to collect/obtain data from website's visitor/user. The information is validated, then sent to the server for further processing and/or storage. Examples of commonly used forms are registration/sign-up forms and login/sign-in forms.
